## Start the app

To start the development just enter the following commands in your terminal : 
````
npm install
npm start
````

# Revents App

Revents App is an application is a social media plateform with a live feed that allows you to create, read, update, and delete a list of events you plan to do during your week. It'a a frontend application
powered by React.js

## Usage
You can basically perform any CRUD operation of your choice in order to manipulate the events.
